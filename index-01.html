Vamos entender JavaScript usando uma lâmpada?

Nesse projeto temos objetivo de executar algumas ações que são:

  • Inicialmente está apagada
  • Quando passarmos o mouse ela acenderá
  • Quando clicar a lâmpada irá quebrar
  • E por fim, ao clicar no ícone de atualizar iremos reiniciar as ações

    • Para isso usei o Javascript para executar essas ações usando condicionais que teve como resultado a estrutura abaixo

    var quebrada = false;
    function mudaLampada(tipo) {
    if (tipo == 1) {
    arquivo = "_imagens/Lampada-acesa.jpg";
    }
    if (tipo == 2) {
    arquivo ="_imagens/lampada-apagada.jpg";
    }
    if (tipo == 3) {
    arquivo = "_imagens/lampada-quebrada.jpg";
    }
    if (!quebrada) {
    document.getElementById("Luz").src = arquivo;
    if (tipo ==3) {
    quebrada = true;
    }
    }
    };
